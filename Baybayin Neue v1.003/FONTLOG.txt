and any derivative works.


Basic Font Information


Copyright © 2022 by John Abila. Hello@JohnAbila.com


Baybayin Neue is a Unicode typeface family that supports the Filipino Tagalog language. This typeface follows the Filipino Baybayin  B17 and B17+  writing standards. 


More specifically, this release supports the following Unicode ranges:
U+1700–U+171F


This release contains 16 font variations:
BaybayinNeue-Black.otf
BaybayinNeue-BlackOblique.otf
BaybayinNeue-Bold.otf
BaybayinNeue-BoldOblique.otf
BaybayinNeue-Light.otf
BaybayinNeue-LightOblique.otf
BaybayinNeue-Medium.otf
BaybayinNeue-MediumOblique.otf
BaybayinNeue-Regular.otf
BaybayinNeue-RegularOblique.otf
BaybayinNeue-Thin.otf
BaybayinNeue-ThinOblique.otf
BaybayinNeue-UltraBold.otf
BaybayinNeue-UltraBoldOblique.otf
BaybayinNeue-Ultralight.otf
BaybayinNeue-UltralightOblique.otf


Documentation can be found at https://www.kickstarter.com/projects/johnabila/baybayin-neue-the-modern-font-of-a-ancient-filipino-script?ref=6bbj9u&token=9c1f4c44

ChangeLog


November 9, 2022 (John Abila) Baybayin Neue Version 1.003
- Third release
- Updated Copyright Information & Description
- Revised naming convention for the 700 "Ultra Bold" font weight
- Revised separator Widths across entire typeface



October 14, 2022 (John Abila) Baybayin Neue Version 1.002
- Second release
- Introduced 4 New Font Weights
- New font weight 100 "Thin" & "Thin Oblique"
- New font weight 900 "Black" & "Black Oblique"
- Revised font weight 700 "Ultrabold" into "Ultra Bold"
- Revised font weight 200 "Thin" into "Ultra Light"
- Fine-tuned vector nodes and extremes throughout the entire typeface
- Updated Copyright Information & Description
- 62+ New Unique Kerning Pairs



August 23, 2022 (John Abila) Baybayin Neue Version 1.001
- Initial release
- Introduced 12 Font Weights 
- Introduced Tagalog Unicode Glyphs “U+1700–U+171F”
- 142+ Unique Kerning Pairs
